Storage & Assets

The Phonographic Management System uses a dedicated, high-performance storage cluster for all binary assets (Audio, Artwork, Legal Documents).

Primary Storage Node

space.phonographicms.com

Asset URLs

When you upload assets or retrieve catalog metadata, file paths will always be returned relative to the storage domain.

Asset TypeURL Structure
Cover Art (Original) https://space.phonographicms.com/images/{release_id}/cover.jpg
Audio Master (WAV) https://space.phonographicms.com/audio/{track_id}/master.wav
Signed Contracts https://space.phonographicms.com/legal/{contract_id}.pdf

Direct Uploads

To upload large files, request a pre-signed URL from the API, then PUT the file binary directly to space.phonographicms.com. 

curl -X PUT "https://space.phonographicms.com/upload/temp/12345?signature=..." \
  --upload-file song.wav
