Introduction

The Phonographic Management System (PMS) API enables partners to provide advanced catalog management, royalty distribution, and rights management functionality directly from their own applications.

All API partners are issued a secure API key. This key is your "Client Secret" and grants administrative access to your tenant.

Security Warning

Your API Key carries high privileges. Never expose it in client-side code (browsers, mobile apps).

Base URLs & Environments

ServiceEndpoint
Production API https://api.phonographicms.com/v1
Asset Storage https://space.phonographicms.com
Sandbox API https://sandbox-api.phonographicms.com/v1

Authentication

The API uses Bearer Token authentication. Include your token in the header of every request. 

Authorization: Bearer <YOUR_ACCESS_TOKEN>
Content-Type: application/json
