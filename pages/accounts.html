POST

Sign Up New User

Endpoint: /partner/account/signup

Use the signup resource to create new PMS child accounts. This action initializes a new tenant, a default Payee, and the Master Rights Contract.

Request Body

ParameterTypeDescription
email * string Unique email for the account owner.
enterpriseName * string Legal name of the Label or Artist entity.
type * string 'Indie' (Single Artist) or 'Distributor' (Multi-Label).

Example Request

curl -X POST "https://api.phonographicms.com/v1/partner/account/signup" \
-H "Content-Type: application/json" \
-d '{ 
  "email":"[email protected]",
  "enterpriseName": "Cosmic Soundwaves LLP",
  "type":"Distributor",
  "partnerApiKey":"sk_live_..."
}'
